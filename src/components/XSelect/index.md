# XSelect

下拉选择器。与原版 [Select](https://ant.design/components/select-cn/) 相比：

- **属性**：新增必需属性 `options` 设置选项列表，简化使用。
- **组件**：新增 `XSelect.Multiple` 和 `XSelect.Tags` 组件提供类型友好的多项选择和标签选择。
