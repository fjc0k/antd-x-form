# XSwitch

开关选择器。与原版 [Switch](https://ant.design/components/switch-cn/) 相比：

- **组件**：将几个常用的选项内置为单独的组件：`是/否`、`开/关`、`开启/关闭`；
- **属性**：增加 `value` 作为 `checked` 的别名，`defaultValue` 作为 `defaultChecked` 的别名，便于在表单中直接使用。
