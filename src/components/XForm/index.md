# XForm

类型完备的表单控件。与原版 [Form](https://ant.design/components/form-cn/) 对比：

- **TypeScript**：类型非常完备，满满的安全感；
- **表单校验**：引进短小精悍的 [Yup](https://github.com/jquense/yup) 做数据校验，同时归拢校验规则；
- **组件**：增加 **FormConditionItem** 作为根据条件不同而渲染不同的表单项。
